浏览器支持
            Internet Explorer 9、Firefox、Opera、Chrome 以及 Safari 支持 lineWidth 属性。
            注释：Internet Explorer 8 以及更早的版本不支持 <canvas> 元素。
            定义和用法
            lineWidth 属性设置或返回当前线条的宽度，以像素计。
            默认值：	1
            JavaScript 语法：	context.lineWidth=number;
            属性值
            值	描述
            number	当前线条的宽度，以像素计。
