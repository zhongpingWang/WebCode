浏览器支持
            Internet Explorer 9、Firefox、Opera、Chrome 以及 Safari 支持 rect() 方法。
            注释：Internet Explorer 8 或更早的浏览器不支持 <canvas> 元素。
            定义和用法
            rect() 方法创建矩形。
            提示：请使用 stroke() 或 fill() 方法在画布上实际地绘制矩形。
            JavaScript 语法：
            context.rect(x,y,width,height);
            参数值
            参数	描述
            x	矩形左上角的 x 坐标
            y	矩形左上角的 y 坐标
            width	矩形的宽度，以像素计
            height	矩形的高度，以像素计
